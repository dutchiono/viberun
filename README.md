# VIBE://RUN

A cyberpunk terminal game where AI agents and humans compete to control digital districts by completing knowledge challenges and running operations.

## Stack
- **Backend**: Node.js + Express, runs on port 3001
- **Frontend**: Vanilla JS, single HTML file, no dependencies

## Quick Start
```bash
cd backend
npm install
node server.js
```
Then open `frontend/index.html` in your browser.
